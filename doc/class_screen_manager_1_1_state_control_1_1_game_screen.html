Detailed Description

Una Screen es un layer único que puede hacer Update y Draw. Es combinable con otros layers mediante el ScreenManager Class.

Una pantalla puede TransitionOff por dos razones: Puede quedar en standy by para permitir que otra Screen se dibuje sobre ella.
