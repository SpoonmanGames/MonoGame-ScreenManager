Detailed Description

ScreenManager es un componente que maneja una o más instancias de GameScreen. Las mantiene en una cola y realiza Update y Draw én el orden en que están guardadas. Mantiene también el input en la pantalla correcta y en standby las pantallas que no están activas en un momento dado.

Esta es la clase principal del proyecto, se extiende de DrawableGameComponent por lo que debe ser implementada al cmienzo de un proyecto Monogame.
