Detailed Description

Lee el input del teclado y del Gamepad manteniendo un estado actual del controller y el estado previo. También controla las acciones generales como por ejemplo "PAUSE" o moverse por el menú

Esta clase permite manejar Inputs ya sea por teclado o por GamePad.
