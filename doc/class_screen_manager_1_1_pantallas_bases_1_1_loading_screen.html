Detailed Description

Cordina la transición entre cambios de contextos. Usado cuando es necesario borrar todas las Screen de la cola y se necesit cargar otra pantalla con mucha carga. Para esto se hace:

  • Le dice a todas las Scren que hagan TransitionOff.
  • Activa la Loading Screen.
  • Loading Screen espera que la Screen previa terminé su transición.
  • También espera que la siguiente screen en la cola cargue su contenido.
