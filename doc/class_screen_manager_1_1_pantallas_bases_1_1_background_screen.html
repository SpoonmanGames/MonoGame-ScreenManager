A diferencia de otras Screen, esta pantalla no debe hacer TransitionOff cuando es tapada por otra. La idea es que siempre se mantenga en el fondo. Por ello el override fuerza o engaña al update para que nunca sepa que tiene otra Screen tapandola. Esto también permite que Background tenga animación inclusive si tiene otras Screen encima.
